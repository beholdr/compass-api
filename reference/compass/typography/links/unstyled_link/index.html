Compass Unstyled Link

Makes a link appear like regular text. Not supported by IE 6 or 7.

This file can be imported using: @import "compass/typography/links/unstyled-link"

@mixin unstyled-link {
  color: inherit;
  text-decoration: inherit;
  cursor: inherit;
  &:active, &:focus {
    outline: none;
  }
}
