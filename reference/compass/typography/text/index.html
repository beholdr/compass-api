Compass Text

Utilities for managing text.

This file can be imported using: @import "compass/typography/text"

Imports

  1. Ellipsis – Text truncation with ellipsis.
  2. Force Wrap – Wrap URLs and long lines of text.
  3. No Wrap – Remembering whether or not there's a hyphen in white-space is too hard.
  4. Text Replacement – Replace text with images.
